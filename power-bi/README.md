# Power BI Package

No `.pbix` is included for this project yet. Power BI Desktop is installed, but saving a new `.pbix` requires a manual GUI build; I did not create a placeholder file.

Included:

| File/folder | Purpose |
|---|---|
| `../data/powerbi/` | Power BI-ready CSVs generated by `python scripts/run_sql.py`. |
| `dashboard_brief.md` | Dashboard purpose and pages. |
| `data_model.md` | Tables and recommended relationships. |
| `dax_measures.md` | Measures to create. |
| `manual_build_instructions.md` | Rebuild guide. |
| `refresh_instructions.md` | Refresh workflow. |
| `screenshots/page*_mockup.png` | Static mockups from included data, not Desktop screenshots. |

The dashboard should translate forecasting performance into an inventory decision: where demand is hardest to predict, where reorder-point value is highest, and which model gives the best operating-cost tradeoff.
